# **App Name**: AdPilot AI

## Core Features:

- Meta Account Connection: Securely connect to a user's Facebook account, retrieve available ad accounts and pages, and store access tokens.
- Automated Ad Launch: A 4-step wizard for launching Facebook Message Ads. Upload video, select the facebook page, specify the number of ads, and launch.
- AI-Powered Ad Copy Generation: Generates multiple ad copies (primary text, headlines, CTA) in both Thai and English, driven by user's requirements.
- Campaign Dashboard: Display campaign performance metrics like spend, messages, and cost per message.
- Automated Ad Optimization: A scheduled process to pull campaign insights, pause underperforming ads, and identify winning ads based on predefined rules and LLM based suggestions. LLM will act as a tool and advise on possible actions but ultimately adhere to defined rules.
- Emergency Pause: Allows immediate pausing of all active campaigns.
- Logging and Auditing: Comprehensive logging of all Meta API interactions and AI decisions.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to evoke trust and efficiency, reflecting the app's functionality.
- Background color: Light gray (#F5F5F5) for a clean and professional look.
- Accent color: A bright green (#90EE90) for success indicators and call-to-action buttons.
- Body and headline font: 'Inter', a sans-serif font, chosen for its modern and neutral appearance suitable for both headlines and body text.
- Use clean and minimalist icons to represent actions and status within the platform.
- Employ a clean, card-based layout for the dashboard and campaign details, ensuring ease of navigation and data comprehension.
- Incorporate subtle animations to provide feedback on user actions and guide the user through key workflows.
